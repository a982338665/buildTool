

**1.linux安装nodejs：**

    1.安装前提：node及npm - 见node文件夹
    2.卸载安装：
        npm uninstall -g @angular/cli   --卸载之前的版本
        npm cache clean --force         --清除缓存，确保卸载干净
        ng -v                           --检查是否卸载干净
        npm install -g @angular/cli@6.0.5   --安装指定版本
        ng -v                           --检查版本号
