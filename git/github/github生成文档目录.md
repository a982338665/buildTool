## 安装使用
    
    1.下载位置 D:\install-soft\gh-md-toc.windows.386
    2.解压后文件：gh-md-toc
    3.重命名：toc.exe
    4.添加环境变量：D:\install-soft\gh-md-toc.windows.386\
    5.cmd窗口执行：
        toc xxx.md
